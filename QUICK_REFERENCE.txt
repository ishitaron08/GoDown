🔐 SECURITY FEATURES
════════════════════════════════════════════════════════════════════════════════

✅ AI Chatbot Protection
   • Prompt injection detection (30+ keywords blocked)
   • System prompt protection (cannot be extracted)
   • Admin/Manager access only
   • Input sanitization + output filtering
   • Suspicious activity audit logging

✅ Role-Based Access Control
   • New users default to Customer role
   • Customers cannot access AI or dashboard
   • Staff cannot use AI features
   • Error messages don't expose secrets

✅ Mobile Responsive
   • Desktop: Credentials panel on right
   • Mobile: Credentials drawer at bottom
   • Auto-fill on click
   • All UI optimized for all devices
